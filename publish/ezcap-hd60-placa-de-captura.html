# Descrição do produto

Ezcap261M HD60 Game Live 
Essa placa não grava direto para HD ou Pen Drive e a USB 3.0 só serve para conectar no PC ou celular, também recomendo que seu PC tenha porta USB 3.0, caso contrário poderá apresentar LAG (atrasos e travamentos) ou perda na qualidade do sinal!

Placa Plug and play, não há necessidade de instalar o driver, basta usar o software para capturar o seu vídeo hd.
Gravação de vídeo de alta qualidade, para captura de vídeo 4k e 1080 p.
Usb 3.0 interface de alta velocidade, poderoso para obter vídeo hd em você computador até 4k.
Com entrada de microfone, você pode gravar o áudio do microfone através dele para que você não perca sequer a narração.
É compatível com para windows, para mac ou para linux.

- O padrão UVC (USB VÍDEO CLASS), não há necessidade de driver, suporta a maioria os softwares de captura e streaming de terceiros. Funciona em qualquer sistema operacional e programa de transmissão de terceiros, não terá problema de incompatibilidade encontrado na placa de captura 284/295. Como ela possui o padrão UVC, você poderá ligar várias placas Ezcap 261M, ao mesmo tempo, mas é importante salientar que seu PC deve ter a quantidade de portas USB 3.0 necessárias para tal finalidade, pois caso utilize HUB USB, mesmo que seja um HUB 3.0 ela poderá apresentar “LAG” e atrasos.
